Average

Type 1: Average of numbers

Explanation:

  1. Now how do I say this.....so basically we were given a number series where we have to find something like average, missing part of the sum, comparing two averages etc. I think this type is the weakest in the averages or so. Okay without wasting time let's dive into some problems.

  2. 1. What is the average of following set of numbers : 302, 152, 132, 122, 112.

    1. 140.
    2. 154.
    3. 160.
    4. 164.

  3. Answer :

    Here we are going to use the formula Average = sum of values / total no.of values
    Step 1: ( 302 + 152 + 132 + 122 + 112 ) / 5
    Step 2: 820 / 5
    Step 3: 164
    The correct answer is 164.

  4. 2. The average of five consecutive numbers is 27 what is the largest number?

    1. 29.
    2. 30.
    3. 28.
    4. 27.

  5. Answer :

    Here we are going to use the formula Average = sum of values / total no.of values

    Note : Consecutive means numbers one after another.


    Step 1: ( (n-2) + (n-1) + n + (n+1) + (n+2) ) / 5 = 27 (average)
    Step 2: 5n / 5 = 27
    Step 3: n = 27 (both 5 got cancelled eachother)
    Step 4: (n+2) = 29
    The correct answer is 29.

  6. 3. The average of four numbers is 20, and the average of six other numbers is 30. what is the average of all 10 numbers?

    1. 25.
    2. 26.
    3. 27.
    4. 28.

  7. Answer :

    Here we are going to use the formula Average = sum of values / total no.of values
    Step 1: ( n1 + n2 + n3 + n4 ) / 4 = 20 (average - 1)
    Step 2: ( n1 + n2 + n3 + n4 ) = 80
    Step 3: ( m1 + m2 + m3 + m4 + m5 + m6 ) / 6 = 30 (average - 2)
    Step 4: ( m1 + m2 + m3 + m4 + m5 + m6 ) = 180
    Step 5: 80 + 180 = 260
    Step 6: 260 / 10 = 26
    The correct answer is 26.

Percentage

Type 1: Percentage on numbers

Explanation:

  1. I thought that "Percentage" is an independent topic to learn on the first day, But it turns out to be dependent on fractions and mixed fractions. I'm some what weak on fractions so I'll make another article that covers fraction separately. Okay without wasting time let's dive into some problems.

  2. 1. A student scored 60% in an exam of 200 marks. what is the mark obtained by the student?

    1. 100.
    2. 120.
    3. 130.
    4. 150.

  3. Answer :

    Here we are going to use the formula Percentage * value
    Step 1: 60% * 200
    Step 2: (60 / 100) * 200
    Step 3: 60 * 2 = 120
    The correct answer is 120.

  4. 2. A Product is sold at a profit of 25%. If the cost price of the product is 400, what is the selling price?

    1. 480.
    2. 500.
    3. 450.
    4. 520.

  5. Answer :

    Here we are going to use the formula Percentage * value
    Step 1: 25% * 400
    Step 2: (25 / 100) * 400
    Step 3: 25 * 4
    Step 4: 100
    Step 5: original price + profit.
    Step 6: 400 + 100.
    The correct answer is 500.

  6. 3. A fruit seller had some apples. He sells 40% apples and still has 420 apples. How many apples he originally had?

    1. 588.
    2. 600.
    3. 672.
    4. 700.

  7. Answer :

    Here we are going to use the formula Percentage * value
    Step 1: x is the original no.of apples
    Step 2: 0.6x = 420
    Step 3: (0.6x / 0.6) = (420 / 0.6)
    Step 4: x = 700
    The correct answer is 700.

    Note : Percentage is simply a fraction of 100.


    1. 100% = (100 / 100) = 1
    2. 60% = (60 / 100) = 0.6
    3. 100% = (60% / 60%) = 1
