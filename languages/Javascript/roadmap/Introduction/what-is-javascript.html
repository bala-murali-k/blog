What is javascript?

Here we are just looking at the introduction of javascript not the history (will be learned later on).

The javascript is a light weighted, interpreted, single threaded, dynamically typed scripting language. It is the core technology of the web. It makes the internet more engaging and alive by adding interactivity to the websites.

Javascript's strength lies in it's ability to run in a browser allowing developers to create applications that are interactive and more engaging web application by,

  1. Adding Interactivity to the website
  2. Create API
  3. Communicate between websites
  4. Manipulating DOM
  5. Manipulating Styles of the Website
  6. Manipulate Strings/Data from the user
  7. Store data inside the browser
Other than these web related uses we can also create games, Mobile apps and Desktop apps. The javascript can be executed by an special program called javascript engine which is present in all browser which make the javascript more popular as it can run in almost all devices.
