  2. Question 1:

    What is the difference between java and javascript?

    Javascript is a client side scripting language and java is a object oriented programming language. Both are so much different compared to each other.

    Javascript :
    It is a dynamically typed, interpreted, light weighted scripting language used for developing interactive websites. Javascript is also known as "Browser Language".

    Java :
    It is a statically typed, compiled, object oriented programming language that has a virtual machine, which allows you to run the program on nearly every platform. Java promised "Write once, Run anywhere".


  3. Question 2:

    What are the different datatypes present in javascript?

    There are three major categories of data type in javascript, they are

    Primitive :

    The primtive data types are Numbers, Strings, Boolean, Symbol.
    Numbers: It basically numbers like 1, 435, 3416, etc.
    String: A String is a collection of characters. A character is just a single letter or symbol. It is recommended to write the string using double quotes but it can also be written using single quotes. Example of creating a string variable: let var1 = "Javascript Interview Questions".
    Boolean: It holds either "true" or "false".
    Symbol: It is immutable (cannot be changed) and unique. We can use it as a key in a object but we cannot loop it using for in loop.

    Composite / Non Primitive :

    The Composite data types are Objects, Functions, Arrays.
    Object: The object is an collection of properties. A property is a key: value pair where the key is a string and value can be anything.
    function: Afunction is a block of code that is designed to perform a specific task.

    Trivial :

    Trivial data types are undefined and null.
    Undefined: This is a type of error when we try to access a variable that is not defined or declared.
    Null: This is the placeholder for a value that indicates the absence of a value. Basically a variable that holds null value is empty.


  4. Question 3:

    How to comment in javascript?

    The comments are like a guide to programmers, simply put it is an understandable description of what the code is doing, where we can use it for our future self or next developer. The compiler will always ignore the comments when running. There is actually a There are two types of comment one is one line comment and the other one is multiline comment.

    Single line: We can comment a single line using "//..." in javascript. Replace the ... with the actual message.
    Multi line: We can comment multi lines using "/*...*/" in javascript. Replace the ... with the actual message.


  5. Question 4:

    What is the use of isNaN function in javascript?

    The isNaN function is used to check if the variable or input is number or not. If the input is number the function will return false and if the input is not a number the function will return true. It is not considered a reliable way of finding if a variable is a number or not. Rather than we can use Number.isNan() for it.

    What are undefined and undeclared variables?

    Undefined: A variable that is declared and is not assigned any value before accessing it. Undefined is not a keyword.
    Example: typeof(variable)
    The result will be undefined.

    Undeclared: Accessing a variable that is not even decalred.
    Example: console.log(variable)

  10. Question 9:

    What are global variables? How are these variables declared, and what are the problems associated with them?

    Global variables are variables that define outside the functions. These variables have a global scope, it means the global variables can be accessed by any function without passing it to them as parameters. It is difficult to debug and test the code that relies on global variable.
