How to open files application using Terminal

In this article we are going to know how to open file manager in linux using Terminal.

Step 1: Open the terminal using the shortcut "ctrl + alt + t.

Step 2: Run the command xdg-open /home.

This command will open your new file manager in the home directory where you can see the profile/account folders on your linux system.

You can also use xdg-open /home/linux(profile/accout - linux in my case) to open directly to your profile just like you freshly open a file manager using the GUI.

I used it on debian and it worked, But I don't know about the other types like arch or red hat and so on.
