How to create a default directory using Java

Required Concepts to be known.

  1. Objects
  2. Constructors
  3. Files

What is this application?

We are going to develop a command line to-do application using java, Where we can add tasks, Change a status of a task, Automatic status update and so on. Inshort just an normal to-do application by ourselves.


In this article we are going to create a default directory for our application based on our operating system. These are the default folders we are going to create in this article.

    For Windows      C:/Users/windows/.todo-list/
    For Linux           home/username/
    Others                 ./todo-list/

As for mac, I didn't have worked with mac so you can replace the directory with your own. Now let's dive into the code. 

						
		import java.io.File;

		class Todo {

			//variables necessary for directory creation
			private String os = System.getProperty("os.name");
			private String file_path = null;
			private String file_path_windows = "C:/Users/windows/.todo-list/";
			private String file_path_linux = System.getProperty("user.home") + "/.todo-list/";

			//Constructor
			Todo() {

				if (os.contains("win")) {
					file_path = file_path_windows;
				} else {
					file_path = file_path_linux;
				}

				//Attempting to create default folder
				createFolder(file_path);

				if (!new File(file_path).exists()) {
					file_path = "./todo-list/";
					createFolder(file_path);
				}
			}

			//method to create folder
			public static void createFolder(String path) {
				File directory = new File(path);

				if (!directory.exists()) {
					if (!directory.mkdirs()) {
						System.err.println("Failed to create directory in " + path);
					} else {
						System.out.println("Directory created successfully in " + path);
					}
				}
			}

			//main method
			public static void main (String[] args) {
				Todo todo = new Todo();
			}
		}

Let's go through this code line by line.

In the line (1) import java.io.File; We are just importing the "File" class with this line as we are going to create the directory with the help of this file class.

In the line (3) class Todo {} we are creating a class named Todo (Make sure that the class is properly closed.)

In the line (6) private String os = System.getProperty("os.name"); we are creating a string variable that stores the name of the operating system you are currently using. You can test it out by System.out.println(System.getProperty("os.name")); This line will print the name of the operating system you are using.
This variable is declared private so that the value is not accessible outside of the Todo class, But it can be accessed and modified within the Todo class.

In the line (7) private String file_path = null; we are creating a variable named file_path and initializing it with the null value so that the compiler does not throw any error while compiling.

In the line (8) private String file_path_windows = "C:/Users/windows/.todo-list/"; we are creating a variable that stores a default path for windows operating system.

In the line (9) private String file_path_linux = System.getProperty("user.home") + "/.todo-list/" we are creating a variable that stores a default path for Linux operating system. The line System.getProperty("user.home"); will return the linux home path.

In the line (12) Todo() {} we are creating a constructor for our class. This block of code will run automatically when an object is created for this class (The object is created in the main method.). Now we are going to look inside the constructor block.

In the line (14) if (os.contains("win")) {} we are checking if the value of the os variable contains "win" the if block will execute. Otherwise the else block will execute.

Inside the if block (15) file_path = file_path_windows; we are updating the file_path with the windows path value.

Inside the else block (17) file_path = file_path_linux; we are updating the file_path with the linux path value incase the if condition is false.

In the line (21) createFolder(file_path); we are calling the createFolder method with the updated file_path variable, which will create the folders with the file_path value.

In the line (23) if (!new File(file_path).exists()) {} we are checking the directory in the file_path variable is created or not. The condition new File(file_path).exists() will be true if the directory in the file_path variable exists, Basically it means if this condition is false then the mission is failed successfully (-100000 Aura) . But we need to do something if the directory is not created so we invert the result using "!" logical NOT operator and create a directory where the program is running.

In the line (24) file_path = "./todo-list"; we are updating the file_path variable with the current folder.

In the line (25) we are again trying to create a folder by calling the createFolder method. That's all with the constructor.



Now we are past the constructor and In the line (30) public static void createFolder(String path) {} we are creating a method that require a string as a argument. Without the string the method won't work. Now we are going to enter inside the createFolder method.

In the line (31) File directory = new File(path); we are creating an object named directory for the class File which we imported in line (1).

In the line (33) if (!directory.exists()) {} we are doing the same in the line (13) and check if the path is available or not. If the directory is available the rest of the code will not run (There is no reason tp create a directory when there already is one). If the directory is not present we will go to the next line.

In the line (34) if (!directory.mkdirs()) {} we are creating the directory in the provided path. The line directory.mkdirs() will create the specified path for us and as for the condition, if the directory is not created we will print the error message in the next line.

In the line (35) System.err.println("Failed to create directory in " + path); we are telling the user that the program fails to create a directory in the machine (OS).

The next lines you know that if the directory is created we are printing a success message to the user. Now we are out of the createFolder method.



In the line (43) public static void main(String[] args) {} we are creating the main method for out application.

In the line (44) Todo todo = new Todo(); we are just creating the object which will automatically execute the above codes and create a folder in the suitable location according to the operating system you are using.
